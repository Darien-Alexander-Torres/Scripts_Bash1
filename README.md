# Primeros Pasos en Bash Scripting - Fundamentos de Automatización

Este repositorio contiene una colección de scripts desarrollados durante mis primeras etapas de aprendizaje en la carrera de **LSTI**. Representan mis ejercicios iniciales de lógica de programación y automatización de tareas básicas en entornos Linux.

#  Descripción de los Módulos

# 1. Gestión de Parámetros y Lógica de Usuario (T16.sh)
Un script enfocado en el manejo de argumentos desde la línea de comandos.
* Funcionalidad: Evalúa si se han pasado parámetros al ejecutar el script y proporciona retroalimentación dinámica al usuario basándose en la cantidad de argumentos recibidos.

# 2. Estructuras de Control e Iteración (T17.sh)
Implementación de lógica condicional y bucles para la toma de decisiones automatizada.
* Funcionalidad: Utiliza estructuras `if-else` y ciclos para procesar información y mostrar resultados en consola de forma organizada.

# 3. Scripts de Utilidad General (Bash 1, 2, 3)
Ejercicios prácticos de interacción con el sistema operativo:
* Bash 1: Automatización de comandos básicos de sistema para consulta de información.
* Bash 2: Manejo de variables y operaciones lógicas simples.
* Bash 3: Interacción con el usuario mediante entradas por teclado (`read`) para personalizar la salida del script.

#  Evolución Técnica
Estos archivos sirven como base técnica para mis proyectos actuales. Demuestran el dominio de:
* Sintaxis básica de **Shell Scripting**.
* Manejo de variables de entorno y parámetros de posición.
* Control de flujo y validación de entradas de usuario.
